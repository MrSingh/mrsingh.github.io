# Basic Git Commands

## Add remote repository

```git
git remote add origin https://github.com/user/repo.git
```

## Check Remote Repository

```git
git remote -v
```

## Stage All Changes

```git
git stage -all
```

## Change Current Branch Name

```git
git branch -m <newname>
```

## Change Any Branch Name

```git
git branch -m <old-name> <new-name>
```
